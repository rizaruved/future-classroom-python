Print() Function



In Python, the print function is a fundamental tool for displaying information. It allows you to output text or variables to the console, making it visible to the user or the developer during program execution. The basic syntax of the print function is straightforward:

Code:

                            
print("Hello, World!")

Output:

Hello, World!

As you can see on the console, you will get the text that you put into the quotation mark. Note that you have to follow the syntax rules which are:

  • The print keyword must be lower case.
  • The print keyword must be followed by opening and closing brackets ()
  • Put any text you want inside a quotation mark within the brackets. The quotation marks can either be single or double.
